# Array flat

### How to use
1. Project setup
  - clone project
  `git clone https://github.com/mat-dudziak/array-flat.git`
  - navigate to cloned project
  `cd array-flat`
  - install dependencies
  `npm i`
  - execute tests:
    - on you current setup
      `npm run test`
    - via Docker
      - `docker compose up`

### About
Suggested node version: 
Logic is located at `./src/array-flat`
